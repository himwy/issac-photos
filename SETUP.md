# Photography Portfolio - Clean Design

## What I Built

A clean, minimalist photography portfolio inspired by BIALONS aesthetic with:

### Design Features
- **Masonry Layout**: Pinterest-style scattered grid with varying image sizes
- **Large Header**: "BIALONS" branding with clean navigation
- **Minimal Aesthetic**: Off-white background (#fafafa) for softer look
- **Smooth Animations**: Framer Motion for elegant page transitions
- **Responsive**: Works perfectly on mobile and desktop

### Pages
1. **Homepage** (`/`)
   - Hero with name and tagline "creative duo"
   - Masonry grid portfolio with mixed aspect ratios
   - Social links in footer

2. **About** (`/about`)
   - About section with bio
   - Contact section with email, phone, social
   - Combined on one page as requested

3. **Project Pages** (`/project/[id]`)
   - Project details with title, year, category
   - Image gallery
   - Back to portfolio link

### Header
- Fixed header that stays visible
- Large "BIALONS" text (change to "ISSAC" if you want)
- Navigation: Portfolio, About, Contact
- Uppercase, tracking-wider styling

### Libraries Installed
- ✅ Framer Motion (animations)
- ✅ react-masonry-css (Pinterest-style layout)

### To Customize
1. Change "BIALONS" to your name in all 3 page files
2. Replace placeholder images
3. Update contact info (email, phone, social)
4. Modify project data
5. Update bio text in About page

### Color Scheme
- Background: #fafafa (off-white)
- Text: #000000 (black)
- Accents: Gray scale
- Clean and minimal

The site now has that artistic, scattered portfolio look!
